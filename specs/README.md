# Vega-Lite JSON specifications

This folder contains human-readable Vega-Lite JSON descriptions for the ten diagrams and maps used in the visualisation.

The web page renders the charts from JavaScript modules in `js/`. Some charts compute derived values first, such as CPI growth, household burden, or the final pressure index. For those charts, the JSON files document the Vega-Lite structure and the derived fields expected at render time.
